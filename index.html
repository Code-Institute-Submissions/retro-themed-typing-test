How Fast Can You Type?

Try out the typing test, clicking play will take you to the test.

Instructions

The test is designed to test how fast you can type there is multiple difficultys to chose from these range from "Easy" to "Typing Pro".
  • 1. Navigate to Typing Test page
  • 2. (Optional) Select the difficulty from the dropdown to the right of the test
  • 3. (Optional) Chose the font colours you would like to use
  • 4. (Optional) Pick how long you would like the test to last for
  • 5. Read the first word which is being presented, click into the typing input box and start typing away!
